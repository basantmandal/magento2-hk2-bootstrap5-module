# Changelog

All notable changes to this module are documented in this file.

The format follows Magento Marketplace and semantic versioning guidelines.

---

## [3.0.0] – 2025-01-26

### Added

* Support for **Bootstrap 5.3.8**

### Changed

* Refactored CDN asset handling for improved reliability and performance.
* Improved JavaScript and CSS asset injection using Magento PageConfig APIs.
* General code quality improvements and internal refactoring.

---

## [2.1.0] – 2024-12-25

### Added

* Support for **Bootstrap 5.3.3**
* Hindi language translation

### Changed

* Updated backend configuration structure and admin tabs
* Updated language and localization files

---

## [2.0.0] – 2023-08-09

### Added

* Support for **Bootstrap 5.2.3** and **Bootstrap 4.x**
* Demo frontend pages for Bootstrap validation
* Localization and translation support

### Changed

* Improved admin configuration options and UI

---

## [1.0.0] – 2022-06-14

### Added
