## 400 Charater

Magento 2 extension to easily add Bootstrap 4 & 5 via CDN without modifying themes. Admin-configurable, CSP-compliant,
debug-ready, and includes demo links. Requires HK2 Core for manual installation. Quickly integrate Bootstrap for
styling, grids, and components in your storefront.

## Marketplace Readiness Summary

* Magento 2.4.x compatible
* CSP-compliant (`script-src`, `style-src`, `font-src`, `connect-src`, `img-src`)
* No inline JavaScript
* Composer-installable
* Privacy disclosure included
* Admin-configurable
* No hidden network calls
* Non-intrusive to theme and UI
